Ox is a lightweight, type-safe foundation for Ethereum software — ABIs, signatures, RLP, transaction envelopes, and more, as composable modules you import only what you need.

Read the docs Source on GitHub 
$ npm i ox
$ pnpm add ox
$ bun add ox

EVM primitives, one import away.

A categorical, module-driven API. Named imports stay fully tree-shakable, so your bundle only includes what you reach for.

Abi

Encode, decode, and parse Application Binary Interfaces with full type inference. 

import { Abi } from 'ox'

const abi = Abi.from([
  'function transfer(address to, uint256 amount) returns (bool)',
])

const data = Abi.encode(abi, 'transfer', ['0x70997…', 1000n])

Address

Validate, checksum, and derive Ethereum addresses from public keys. 

import { Address } from 'ox'

const address = Address.from('0x70997970c51812dc3a010c7d01b50e0d17dc79c8')

Address.checksum(address)
Address.validate(address)

Bytes

Convert between byte arrays, strings, numbers, and hex with predictable semantics. 

import { Bytes } from 'ox'

const bytes = Bytes.fromString('hello world')

Bytes.toHex(bytes)
Bytes.toString(bytes)

Hex

Hex string utilities: pad, slice, concat, and convert to and from any primitive type. 

import { Hex } from 'ox'

const hex = Hex.fromString('hello')

Hex.padLeft(hex, 32)
Hex.slice(hex, 0, 4)

Rlp

Recursive Length Prefix encoding and decoding for Ethereum's serialization format. 

import { Hex, Rlp } from 'ox'

const encoded = Rlp.fromHex([
  Hex.fromString('hello'),
  Hex.fromString('world'),
])

Secp256k1

Sign and verify with the curve underpinning every Ethereum account. 

import { Secp256k1 } from 'ox'

const signature = Secp256k1.sign({
  payload: '0xdeadbeef',
  privateKey: '0x…',
})

Signature

Parse, serialize, and recover ECDSA signatures across the formats you encounter. 

import { Signature } from 'ox'

const signature = Signature.from('0x…')

Signature.recoverPublicKey({ payload, signature })

TxEnvelope

Legacy, EIP-1559, EIP-2930, EIP-4844 — build and serialize every envelope. 

import { TxEnvelopeEip1559, Value } from 'ox'

const envelope = TxEnvelopeEip1559.from({
  chainId: 1,
  to: '0x70997…',
  value: Value.fromEther('1'),
})

TypedData

EIP-712 typed data hashing and signing with strict type checks at the boundary. 

import { TypedData } from 'ox'

const hash = TypedData.hash({
  domain,
  types,
  message,
})

JsonRpc

Type-safe JSON-RPC request, response, and error definitions for every method. 

import { JsonRpc } from 'ox'

const request = JsonRpc.Request.from({
  method: 'eth_blockNumber',
})

Provider

EIP-1193 provider wrapper for talking to wallets and nodes with structured errors. 

import { Provider } from 'ox'

const provider = Provider.from(window.ethereum)

const accounts = await provider.request({
  method: 'eth_accounts',
})

WebAuthnP256

Sign and verify with passkeys — first-class WebAuthn support for modern accounts. 

import { WebAuthnP256 } from 'ox'

const signature = await WebAuthnP256.sign({
  challenge: '0x…',
  credentialId,
})
